# Manual Test Data

Hand-verified, high-quality test set for policy-guided RAG.

## Files

- `cards.json` - 5 credit card products with detailed features
- `guidance.json` - 7 policy rules for card recommendations
- `queries.json` - 15 hand-verified test queries with ground truth

## Data Quality

All queries and ground truth labels have been manually verified to ensure:
- Clear guidance applicability
- Unambiguous correct answers
- Representative coverage of policy rules
